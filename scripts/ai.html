Overview
Curriculum

Fundamentals

📖Problem Framing

Begin the journey of building an intuition for the kinds of problems that are well-suited for machine learning.

🏋️Environment Setup

Establish the developer tools, workflows, and system configurations to hit the ground running.

🧑‍🏫Linear Algebra Fundamentals

Understand how vectors and matrices represent data and transformations in high-dimensional spaces.

🏋️Data Tools & Techniques

Gain experience with NumPy, Matplotlib, and Pandas to clean, process, manipulate and visualize data.

🏋️Simple Linear Regression

Train a model to predict house prices using gradient descent, visualizing the cost curve and descent path.

🏋️Building with LLMs

Use LangChain to chain prompts and create an agent that interacts with third-party sources and APIs.

🧑‍🏫Probability & Statistics

Develop an intuition for probability, distributions, and convolutions to reason about model functioning.

🧑‍🏫Intro to Neural Networks

Build a foundational understanding of gradient calculation and backpropagation algorithms.

🧑‍🏫The ML Terrain

Explore the diverse, rapidly evolving ML landscape, considering job roles, breakthroughs, and ethics.

Shallow Learning

🧑‍🏫Spatializing Data

Visualize matrix transformations, dimensionality reduction and expansion, and latent space.

🏋️Vector & Semantic Search

Explore high-dimensional data by using embedding models for semantic comparison and retrieval.

🧑‍🏫Inference & Training Mechanics

Explore the relationship between data features, parameters, and hyperparameters in the training cycle.

🧑‍🏫Model Development Lifecycle

Tour the iterative training process from benchmarking and measurement to optimization and validation.

🏋️Multiple Linear Regression

Develop a prediction model using Weights and Biases, Scikit-learn, and Matplotlib to predict student grades.

🏋️Gradient Boosted Trees

Train a gradient boosted model with XGBoost and tune hyperparameters to predict home sale prices.

🧑‍🏫Taxonomy of Shallow Algorithms

Explore shallow algorithms and their systematic applicability across diverse machine learning tasks.

🏋️Dimensionality Reduction

Use Scikit-learn to apply PCA, t-SNE, and UMAP on MNIST data and visualize the projections.

🏋️Classification & Clustering

Apply K-Nearest Neighbors, Support Vector Machines (SVMs), Logistic Regression, and K-Means clustering.

🧑‍🏫Nondeterministic Paradigm Shift

Contrast the deterministic paradigm of software engineering with the nondeterministic paradigm of ML.

🏋️Anomaly Detection

Use the IsolationForest algorithm to perform anomaly detection on bank transactions to detect fraud.

🎨Module Project

Solve a real-world classification problem by preparing data, engineering features, and optimizing a model.

Deep Learning

🧑‍🏫Neural Network Optimization

Explore gradient descent optimization algorithms and their role in efficiently training neural networks.

🧑‍🏫Anatomy of a Neural Network

Explore neural networks from the bottom-up: input features, weights, activations, embeddings, and layers.

🏋️Deep Neural Networks

Instantiate and train a neural network using PyTorch to optically identify handwritten digits using MNIST data.

🏋️Debugging & Testing

Learn to validate data, debug and optimize models, and monitor performance metrics across an ML pipeline.

🧑‍🏫Layer-Level Architecture

Explore the anatomical structure of layer types like convolutional, dropout, recurrent, pooling, and more.

🏋️Enhancing a Neural Network

Experiment with manipulating the layers of a neural network to enhance accuracy and extend functionality.

🧑‍🏫Taxonomy of Neural Networks

Explore neural network architectures and their systematic applicability across diverse task types.

🧑‍🏫The ML Research Workflow

Explore the iterative process from problem framing and data cleaning to convergence and productionization.

🧑‍🏫System-Level Architectures

Explore multi-network architectures like generative adversarial networks and encoder-decoder networks.

🧑‍🏫Attention & Transformers

Explore the transformer architecture and attention mechanism, which enable context-aware predictions.

🏋️Implementing Attention

Build an attention head and explore weighted dot products, the key components of a transformer model.

🎨Module Project

Pick a time series dataset and build an LSTM-based neural network using PyTorch to forecast values.

Leveraged ML

🧑‍🏫Leveraged ML Strategies

Explore the taxonomy of transfer learning techniques and their respective use cases (e.g., PEFT with LoRA).

🏋️Transfer Learning with Embeddings

Compare the performance of training an image classifier directly on image data vs. using embeddings.

🏋️Transfer Learning with Fine-Tuning

Fine-tune Resnet18 to perform a binary classification task with more accuracy compared to the base version.

🧑‍🏫LLM Overview

Explore the capabilities, limitations, and frontiers of large language models from agents to RLHF, and more.

🏋️Retrieval Augmented Generation

Vectorize a data corpus into a vector database and run a semantic search before inference to enhance output.

📖The Rules of Machine Learning

Explore best practices for reproducibility, performance, data quality, and iterative improvement.

🧑‍🏫MLOps Fundamentals

Explore key principles, tools, and procedures for improving automation, reproducibility, and monitoring.

🏋️Deploying Models to AWS

Use TorchServe to package and deploy an image recognition model for remote, server-based inference.

🎨Module Project

Select a transfer learning, RAG, or LLM API-driven strategy to solve a real problem from end-to-end.
